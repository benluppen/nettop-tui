# NETTOP-TUI

#### This project was vibe coded using Replit as a learning experiment.

## Installation
1. Download or clone the repository
2. `chmod 755 install.sh`
3. `./install.sh`
4. App should install to /usr/local/bin/nettop
5. run the applicaiton with `sudo nettop`

*NOTE: certain applications names will not display in the TUI without sudo.*
